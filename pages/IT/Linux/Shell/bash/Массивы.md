Элементы массива разделяются пробелами.

Для обработки строк, как элементов массива нужно на время изменить разделитель:

```bash
OLD_IFS="$IFS"
IFS=$'n'

declare -a a
a=( $(cat "file.txt") )
echo "Total:" ${#a[@]}
for i in "${a[@]}"
do
  echo "$i"
done

IFS="$OLD_IFS"
```
