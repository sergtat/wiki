# SSH-клиент для Windows с вкладками
Админам, пользующимся Windows на своих рабочих машинах может пригодиться.

Как-то всю жизнь пользовался Putty в качестве ssh-клиента для windows. Но Putty при всех своих прелестях может бесить огромным количеством отдельных окошек в которых легко запутаться. Поэтому нужен клиент с вкладками, как в браузере. Долгое время пользовался для этого клиентом Poderosa. Потом он у меня перестал подключаться (только что справился с ним, значит будет отдельный пост) и я начал искать другие варианты. Вначале нашелся SuperPutty - это оболочка над Putty. Она хоть и создает вкладки, но фактически запускает Putty и затем перехватывает ее. Это сопровождается дерганиями и проблемами с переключением на другие окна. Следующая же попытка оказалась успешной и я напал на отличный ssh-клиент для windows с вкладками.

Собственно им оказалась программа Xshell. Все фичи я тут просто не смогу описать. Их огромное количество. Почитайте на [официальном сайте](http://www.netsarang.com/products/xsh_key_features.html), правда на английском.

Но осветим кое-что вкратце. Понятное дело, вкладки. Быстрое открытие двойным щелчком. Перемещение. Кнопка быстрого доступа к созданию новых соединений. При этом на экране можно оставить собственно строку вкладок и основную рабочую область. Доступ к меню по кнопке Alt или по щелчку правой кнопкой по заголовку окна. Идеально, ничего лишнего. Вот как это выглядит:

![XShell](/images/Windows/xshell-01.png 'XShell')

Что еще может Xshell? В этом ssh-клиенте есть несколько встроенных команд позволяющих создавать новые подключения, открывать существующие, просто подключаться с помощью ssh, telnet, rlogin, sftp, ftp. То есть заодно является простым FTP-клиентом для командной строки. Также позволяет использовать несколько команд Windows. Жаль нельзя новые подключения открывать в новой вкладке. Перечень доступных команд на снимке экрана:

![XShell](/images/Windows/xshell-02.png 'XShell')

Позволяет вешать команды или даже наборы команд на горячие клавиши. Гениальная функция этого ssh-клиента - можно отправить  одну команду на все открытые консоли. Так можно перезагрузить десять серверов за раз :-D Есть блок отображения информации о туннелировании в котором показаны все серверы через которые вы подключены.

Еще интересные функции. Логирование всего в файл. Можно скопировать все в блокнот. Ну и распечатать, если нужно. Есть дополнительная программа для массового управления всеми запущенными окнами, а также простой ssh-сервер.

В общем, сложно придумать что-то лучше для Windows. Для домашнего и учебного использования бесплатно. Итак, [скачать](http://www.netsarang.com/download/down_xsh.html).
