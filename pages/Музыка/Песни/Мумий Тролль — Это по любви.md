# Мумий Тролль — Это по любви
```
Поцелуями…
Нежными ли…
Нижними ли…
Это будет не трудно,
Это по любви.

Караванами, пароходами
Я к тебе прорвусь,
Mon amie.
Рациями, факсами, телефонами.
Пока завязаны мы.
Удивления хочешь,
vis - а - vis…
Это будет не трудно,
Это по любви.

Расстёгнутыми,
Поперёк весны.
Радуется лето,
Радуемся мы.
Фотолюксами проявленными,
Друг друга запомним альбомами.
Солнцами ли,
Звездами ли…
Это будет не трудно,
Это по любви.

Караванами, пароходами
Я к тебе прорвусь,
Mon amie.
Рациями, факсами, телефонами.
Пока завязаны мы.
Удивления хочешь,
vis - а - vis…
Это будет не трудно,
Это будет не трудно,
Это по любви.

Поцелуями… Нежными ли…
Нужно было ли?…
Это было не трудно…
```
